# GithubPagesTest

A test repository for GitHub Pages. Static HTML pages are generated by Python scripts and published automatically to GitHub Pages via a GitHub Actions workflow.

## How it works

1. **Python script** – `scripts/generate_pages.py` generates static HTML pages into the `_site/` directory.
2. **GitHub Actions workflow** – `.github/workflows/deploy.yml` runs on every push to `main` (or manually via *workflow_dispatch*). It executes the Python script and deploys the resulting `_site/` directory to GitHub Pages.

## Run locally

```bash
python scripts/generate_pages.py
# Opens the generated site
open _site/index.html   # macOS
xdg-open _site/index.html  # Linux
```

## Adding pages

Edit the `PAGES` list in `scripts/generate_pages.py` to add or modify pages. Each entry requires a `filename`, `title`, `heading`, and `content` (HTML string).
